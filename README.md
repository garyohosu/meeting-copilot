# 📋 Meeting Copilot

**会議の文字起こし → 議事録・メール・TODOを自動生成するWebアプリ**

Pixel Recorder、Otter.ai、Notta、Microsoft Teams などの文字起こし結果を貼り付けるだけで、AIが議事録・メール文・TODO・決定事項を一括生成します。

## ✨ 機能一覧

| 機能 | 説明 |
|------|------|
| 📝 本文貼り付け | 文字起こしテキストを直接貼り付けて処理 |
| 🔗 URL入力 | 認証不要の公開URLから本文を自動取得 |
| 📊 要約生成 | 会議全体の要点・主要トピック・決定事項・次のアクションを要約 |
| 📄 議事録生成 | 会議名・日時・出席者・議題・決定事項・TODO・未解決事項を含む正式議事録 |
| ✉️ メール文生成 | 会議報告 / 議事録共有 / 依頼メール の3テンプレートに対応 |
| ✅ TODO抽出 | タスク・担当者・期限を構造化して一覧表示 |
| 🎯 決定事項・未解決事項抽出 | 決定したことと未解決の疑問点を分けて表示 |
| 🔄 タブ別再生成 | 各タブに追加指示を入れて個別に再生成 |
| 📋 コピー | タブごとにワンクリックでコピー |
| 💾 Markdown保存 | 全生成結果をまとめて `.md` ファイルでダウンロード |

## 📖 使い方

### ステップ1 — 文字起こしを入力する

アプリを開くと入力フォームが表示されます。

**A. 本文貼り付け（推奨）**

1. 「📝 本文貼り付け」を選択（デフォルト）
2. Pixel Recorder などの文字起こし結果をテキストエリアに貼り付け
3. 文字数カウンターで上限（50,000文字）を確認

**B. URL入力**

1. 「🔗 URL入力」を選択
2. 文字起こし共有URLを入力
3. ※ 認証が必要なURLは取得できません。その場合は本文貼り付けに切り替えてください。

---

### ステップ2 — 会議情報を入力する（任意）

より精度の高い出力のために、以下を入力できます（すべて省略可）。

| 項目 | 入力例 | 効果 |
|------|--------|------|
| 会議名 | `週次定例会議` | 議事録・メールのタイトルに反映 |
| 日時 | `2026-03-15 10:00` | 議事録の日時欄に反映 |
| 出席者 | `田中、鈴木、佐藤` | 議事録の出席者欄に反映 |
| メール種別 | `会議報告メール` | 生成するメール文のテンプレートを選択 |
| 追加指示 | `簡潔にまとめてください` | 全タブに適用される共通指示 |

---

### ステップ3 — 生成する

「✨ 生成する」ボタンを押すと、AIが処理を開始します。  
数秒〜十数秒で結果が表示されます。

---

### ステップ4 — 結果を確認・活用する

生成結果は5つのタブで確認できます。

#### 📊 要約タブ
会議全体の要点を簡潔にまとめた概要です。  
上司への報告や社内共有にそのまま使えます。

#### 📄 議事録タブ
正式な議事録フォーマットで生成されます。

```
# 議事録
## 会議名 / 日時 / 出席者 / 目的
## 議題
## 内容要約
## 決定事項
## TODO / 宿題
## 未解決事項
## 次回予定
```

#### ✉️ メールタブ
タブ上部のプルダウンで3種類のテンプレートを切り替えられます。

| テンプレート | 用途 |
|-------------|------|
| 会議報告メール | 上司や関係者への会議結果報告 |
| 議事録共有メール | 出席者への議事録送付 |
| 依頼メール | 会議で発生した依頼事項の連絡 |

テンプレートを切り替えると自動的に再生成されます。

#### ✅ TODOタブ
会議で発生したタスクが構造化されて表示されます。

```
☐ 資料修正  👤 田中  📅 2026-03-20
☐ 見積もり依頼  👤 鈴木
☐ 次回アジェンダ作成
```

#### 🎯 決定事項タブ
会議で決定したことと、未解決の疑問点・確認事項が表示されます。

---

### ステップ5 — コピー・保存する

**コピー**  
各タブ下部の「📋 コピー」ボタンでクリップボードにコピーできます。

**Markdown保存**  
右上の「💾 Markdownで保存」ボタンで、全タブの内容を1つの `.md` ファイルとしてダウンロードできます。  
ファイル名は `meeting-copilot-{会議名}-{日付}.md` 形式です。

---

### 再生成（追加指示）

各タブの「🔄 再生成」ボタンを押すと、追加指示欄が表示されます。

**使用例:**

| タブ | 追加指示の例 |
|------|------------|
| 要約 | `3行以内でまとめてください` |
| 議事録 | `英語の発言は日本語に翻訳してください` |
| メール | `件名も含めてください` |
| TODO | `優先度（高/中/低）も付けてください` |
| 決定事項 | `より具体的な表現にしてください` |

初回生成時の全体追加指示は引き継がれ、タブ専用の追加指示が上乗せされます。

## 🛠️ セットアップ（ローカル実行）

### 必要なもの

- Python 3.10 以上
- OpenAI API キー

### インストール

```bash
git clone https://github.com/garyohosu/meeting-copilot.git
cd meeting-copilot
pip install fastapi uvicorn openai requests python-multipart
```

### 環境変数の設定

```bash
export OPENAI_API_KEY="sk-..."
export OPENAI_MODEL="gpt-4o-mini"  # 省略時は gpt-4o-mini
```

### 起動

```bash
cd app
uvicorn main:app --host 127.0.0.1 --port 8001 --reload
```

ブラウザで `http://localhost:8001` を開いてください。

## 🔒 セキュリティ・プライバシー

- **APIキーは環境変数で管理**しており、ソースコードに埋め込まれていません
- 入力された文字起こし本文は **OpenAI API に送信**されます
- アプリ自身はサーバー側に本文・生成結果を**永続保存しません**（セッション中のみメモリ保持）
- ログにはエラー概要のみ記録し、本文は含みません
- **機密情報・個人情報を含む会議データ**を扱う場合は、組織内ルールおよびOpenAI利用規約を確認の上、送信可否をご判断ください
